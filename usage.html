설치

“pyhwpx”를 사용하기 위해서는 파이썬 3.10 이상의 버전과 아래아한글이 설치되어 있어야 합니다. 내부적으로 win32com을 사용하므로, 윈도우OS에서만 실행 가능합니다. 설치는 “pip”으로 간단히 진행하실 수 있습니다.

(.venv) C:\Users\User\HwpProject> pip install pyhwpx
